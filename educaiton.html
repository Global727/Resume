I completed my juinor year of college but then decided to pursue computer science. In order to do this I decided that the MIT xPro Professional Certificate in Coding: Full Stack Development with MERN program was the correct choice for me. In this program we learn web development throught the use of the MERN stack, which is Mongo.db, Express.js, React, and Node.js. The corriculum is created and taught by current MIT professors. The program is composed of hours of lectures and mulitple small projects each week and larger projects that wrap up the main languages we learned. we learned foundational skills with Javascript, HTML, and CSS. In the Javascript module we learned things like call back functions, array manipulation, Asynchronous code. In the HTML and CSS module we learned how to utilize bootstrap and frameworks like grid. Then we moved onto React where we were introduced to JSX, and learning to use react hooks to store information using state and learned the scope of parent and child states and much more. Currently we are working on creating a back end running on mongo and utlizing docker to create containers, express routes and GraphQL
