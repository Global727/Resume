Hello, my name is Riley Bickford, I was formerly a finance major at Florida Gulf Coast University. I completed my junior year of college but then decided to pursue computer science. In order to do this I decided that the MIT xPro Professional Certificate in Coding: Full Stack Development with MERN program was the right choice for me. This program teaches web development through the use of the MERN stack. The MERN stack is Mongo.db, Express.js, React, and Node.js. The curriculum is created and taught by current MIT professors John R. Williams, and Abel Sanchez. The program is composed of hours of lectures and multiple small projects each week and larger projects that wrap up the main languages we learned. We learned foundational skills with Javascript, HTML, and CSS. In the Javascript module we learned things like call back functions, array manipulation, and asynchronous code. In the HTML and CSS module we learned how to utilize bootstrap, and frameworks like grid. We then moved onto React where we were introduced to JSX, and learned to use react hooks to store information using state and learned the scope of parent and child states and more. Currently we are working on creating a back end, running on mongo and utilizing docker to create containers, express routes and GraphQL.
